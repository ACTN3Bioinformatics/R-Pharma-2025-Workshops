Asia/Pacific Presentations

13 presentations showcasing regional R innovations

Overview

The Asia/Pacific session featured 13 presentations covering a wide range of topics from open-source culture and data strategy to AI implementation and clinical reporting automation. These presentations showcase the growing adoption of R and innovative applications across pharmaceutical companies in the region.

Strategic & Cultural Topics

Open-Source Culture and Data Strategy in SHIONOGI

Yoshitake Kitanishi (SHIONOGI)

Presentation on creating value through open-source adoption and data strategy implementation at SHIONOGI.

Topics likely covered:

  • Open-source adoption strategy
  • Data-driven decision making
  • Creating organizational value
  • Cultural transformation

Strategically Assisting Statistical Programmers (SAS2R)

KuenHung Lin

Framework for supporting statistical programmers transitioning from SAS to R.

Focus areas:

  • Migration strategies
  • Training approaches
  • Support systems
  • Success metrics

Technical Implementations

Side-by-Side by Design: Pharma Data Handling

Yutaka Morioka, Yuki Nakagawa

Comprehensive comparison of data handling approaches in pharma using merge, join, match, and hash operations in R.

Key concepts:

  • Data merging strategies
  • Join operations optimization
  • Hash table implementations
  • Performance comparisons

Risk Assessment Deep Dive

Ryo Nakaya

In-depth exploration of risk assessment methodologies and tools for pharmaceutical R packages.

Topics:

  • Package risk evaluation
  • Validation frameworks
  • Quality metrics
  • Decision criteria

  • Error reduction
  • Reproducibility

autoslideR: Streamlining Slide Deck Generation

Yolanda Zhou, Joe Zhu (Roche)

R package for automating clinical reporting slide decks (also presented in EU/US sessions).

Key advantages:

  • 0.5-4 days time savings
  • Multiple event types supported
  • Customizable layouts
  • Placeholder functionality
